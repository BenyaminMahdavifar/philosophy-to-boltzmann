01 — Foundations

From tool to philosophy

The Neuron

A processing unit that receives data, transforms it into information, then shares it. Its output always exists — even when it is random or neutral.

The Goal

Not merely discovering pre-existing relationships, but building new relationships from existing ones — a generative, constructive stance.

The Philosophy

Learning by building relationships from relationships. The network discriminates data by constructing internal structure across states.

03 — Dynamics

The continuous information cycle

Even when a neuron "deliberately" avoids discovering a relationship, this cycle holds. The global network dynamics — not individual neurons — build meaningful structure.

Crisis of Void

Under the uncertainty principle, a neuron may produce no meaningful relationship. But because it always emits an output, the cycle never breaks — even a random signal is an input for all connected neurons.

Bandwidth maximisation

Full connectivity means information is never bottlenecked. Every output is simultaneously available to every other neuron — this is the maximum achievable exchange rate for a network of n units.

04 — Convergence

Connection to the Boltzmann distribution

Boltzmann probability
\[ P(\text{state}) = \frac{e^{-E(\text{state}) / T}}{\displaystyle\sum_{\text{all states}} e^{-E / T}} \]

The fully connected, symmetric structure guarantees that with fixed weights and the same input, the network converges to the same equilibrium distribution. Information is redistributed — not lost — to reach minimum energy.

Advantage over SOM
\[ P(\mathbf{x}) \text{ explicit} \implies \text{higher-order statistics captured} \]

05 — Properties

Reproducibility and stability

Reproducibility
Given identical weights and inputs, the fully connected symmetric network always converges to the same equilibrium. Results are deterministic in distribution.
Stability (steady state)
Continuous output exchange drives the network toward a configuration that minimises system energy — the steady state of the Boltzmann model.
Information conservation
Information is never destroyed; it is redistributed across states. This redistribution is both the mechanism and the goal of the model.
Higher-order statistics
Unlike SOM, this model learns both discrimination among data and higher-order statistical relationships — because it maintains an explicit probability distribution.
